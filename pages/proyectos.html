Proyectos

E-commerce

Tienda online desarrollada con React y Node.js que permite a los usuarios explorar productos, agregarlos al carrito de compras y realizar pedidos. Incluye gestión de inventario, autenticación de usuarios y panel de administración.

Sistema de Turnos

Aplicación web para la gestión de citas médicas que facilita la reserva, modificación y cancelación de turnos. Cuenta con notificaciones automáticas, administración de profesionales y visualización de horarios disponibles.

Blog Personal

Plataforma de blogging con autenticación de usuarios que permite crear, editar y publicar artículos. Incluye sistema de comentarios, categorización de contenido y diseño adaptable para dispositivos móviles.

Sistema de Subida de Archivos (PERN Stack)

Plataforma full stack para la carga, gestión y almacenamiento de archivos en la nube. Permite a los usuarios subir documentos, imágenes y otros archivos de forma segura, con autenticación de usuarios y control de acceso.
